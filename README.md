# xyb
以最优雅的方式示例了springboot最简单的例子。
    
    JPA数据持久
    druid数据库连接池  
    统一接口返回（异常拦截与正常返回统一）   
    日期交互优雅处理   
    定时任务   
    注解异步    
    运行监控    
    在线文档    
    注解缓存    
    安全校验    
    注解数据校验    
    redis   
    sharding-jdbc  
    多环境  
    CSRF/XSS拦截    
    日志系统（包含输入输出参数打印）    
    分布式session````    
    常用工具包：二维码，EXCEL导入导出，CSV导入导出，反射工具包，加密工具包    
    tomcat切换到jetty    
    jsonwebtoken    
    拦截器，监听器，过滤器示例。    
    单点登录    
    数据库优雅分页    
    文件上传下载
